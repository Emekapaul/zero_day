This project is about creating a work environment in my local machine.
You can accomplish this using a dual boot or virtual machine.
I created my work environment using a dual boot.
Dual boot means having two operating systems on a pc, in my case a windows os and a linux os.
The advantage of a dual boot is that the operationg system will have full access to the resources of the pc's hardware.
You can only boot one operating system at a time.
